Joshua Gancher

Assistant Professor
Northeastern University
Office: WVH 332

I apply tools from Formal Methods and Programming Languages to construct and certify secure systems -- particularly systems that use cryptography. Broadly, I am interested in applied cryptography, distributed systems, type systems, proof assistants, and compiler correctness.
